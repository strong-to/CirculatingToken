# FooterLogo 组件图片资源说明

## 图片文件列表

- `asterisk.png` - 星号图标，用于品牌展示
- `books.png` - 书籍图标，用于品牌展示
- `dune.png` - Dune 平台 Logo，数据分析平台
- `oas.png` - OAS 平台 Logo
- `opal.png` - Opal 平台 Logo

## 使用说明

这些图片在 FooterLogo 组件中循环滚动展示，用于展示合作伙伴或相关平台的品牌标识。

## 图片规格

- 建议尺寸：200px × 50px
- 格式：PNG（支持透明背景）
- 用途：品牌 Logo 展示
