# 图片资源目录

## 目录说明

此目录用于存放项目的静态图片资源。

## 使用方式

在 Next.js 中，`public` 目录下的文件可以通过根路径直接访问。

例如：
- 图片路径：`public/images/logo.png`
- 在代码中使用：`/images/logo.png`

## 推荐的文件结构

```
public/
  images/
    logo.png          # Header 中的 logo
    favicon.ico       # 网站图标
    ...
```

## 注意事项

1. 图片文件名建议使用小写字母和连字符
2. 图片格式推荐使用 WebP、PNG 或 SVG
3. 大图片建议进行压缩优化
